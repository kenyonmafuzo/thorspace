# Badges Icons

Este diretório contém os ícones PNG das badges do jogo.

## Estrutura

Cada badge deve ter um arquivo PNG com 512x512 pixels (ou maior, mantendo proporção quadrada).

## Lista de Badges Necessárias:

### PROGRESSÃO
- `rookie_pilot.png` - Nível 5
- `veteran_pilot.png` - Nível 15
- `elite_pilot.png` - Nível 30

### VITÓRIAS & COMBATE
- `first_blood.png` - Primeira vitória multiplayer
- `ace_commander.png` - 10 vitórias multiplayer
- `unstoppable.png` - 3 vitórias consecutivas

### ENGAJAMENTO
- `daily_recruit.png` - 3 dias diferentes
- `daily_soldier.png` - 7 dias consecutivos

### ESPECIAIS
- `strategist.png` - Vitória com 3 tipos de nave
- `iron_mind.png` - Vitória em desvantagem

## Formato Recomendado:
- Formato: PNG com transparência
- Tamanho: 512x512px
- Estilo: Ícones espaciais/militares com visual futurista
- Cores: Predominância de azul ciano (#14ffff), dourado, prateado
