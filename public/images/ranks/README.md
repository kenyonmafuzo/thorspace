# Rank Assets

Este diretório contém os ícones de rank do sistema de progressão.

## Estrutura

Cada tier tem 3 materiais (bronze, silver, gold):

```
/images/ranks/
├── rookie/
│   ├── rookie_bronze.png
│   ├── rookie_silver.png
│   └── rookie_gold.png
├── veteran/

├── elite/
```

## Especificações

- **Formato**: PNG com transparência
- **Dimensões**: 512x512 px
- **Fundo**: Transparente
- **Total**: 21 ícones (7 tiers × 3 materiais)

## Notas

- Os ícones devem seguir uma progressão visual clara
- Bronze < Silver < Gold em brilho/qualidade visual
- Cada tier deve ter identidade visual distinta
- Usar cores e símbolos que representem o nível de maestria
